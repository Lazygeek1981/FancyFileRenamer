Fancy File Renamer

A fancy little tool to rename a bunch of files all at once
freeware and open source



Task based file renaming Create and combine different tasks as needed

You can do even the most complex renaming jobs by combining different renaming tasks. For instance use a replace task to replace some characters and than add an enumeration tasks and change the file extension all at once. The possibilities are endless

Sort files before renaming Use various attributes - even EXIF Tags - for sorting

If you want to change the order of the files before you rename them (e.g. if you added an enumeration task) you can choose from different sorting options. Sort files by their original name, size or even use their EXIF tags (if present). See the tutorial video for some examples how to use the sorting task.

See a result preview before renaming The preview is updated in real-time and even indicates if the new filenames are valid

All tasks (sorting and renaming) can be previewed in real-time. So you can check your results while you are adding or editing. The Fancy File Renamer will even check for duplicated file names or invalid file names. Those will be highlighted with a red background colour while valid file name are printed on a green background. So you'll get immediate feedback if all your changes are applicable.
